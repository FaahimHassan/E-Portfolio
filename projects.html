My Projects

C# &.net project(Fresh Farm Shop management)

I developed a desktop-based shop management system to automate inventory and sales operations. The system supports role-based access for Admin and Manager and integrates a database for managing products, customers, and sales transactions.

Features:

Login Module: Secure login system for Admin and Manager.

Admin Dashboard: Admin can add products, manage stock, create manager accounts, and view sales information.

Manager Panel: Managers can view available products and process customer sales.

Product Management: Add, update, and manage product inventory.

Billing / Invoice System: Automatically generates invoices for customer purchases.

Sales Report: Tracks and stores sales data in the database.

Java Project(coffe shop management)

I developed a desktop-based coffee shop management system that allows customers to browse menu items, place orders, apply discount coupons, and complete payments. The system also includes an admin panel for managing menu prices and controlling the application.

Features

Customer Login Module: Customers can register and log in using their email and password before placing orders.

Coffee Menu Interface: Displays different coffee items with images and prices, allowing users to browse the available menu.

Shopping Cart System: Customers can add coffee items to the cart, increase or decrease quantity, and view the updated total price.

Promo Code / Discount Feature: Users can apply coupon codes to receive discounts on their orders.

Payment Module: Supports digital payment options such as bKash, Rocket, and Nagad for completing the purchase.

Admin Panel: Admin can log in and update the prices of coffee items in the menu.

Order Confirmation System: After successful payment, the system generates an order confirmation with a unique token number.

Token Display Page: Displays a confirmation message and token number for customers to collect their order.
